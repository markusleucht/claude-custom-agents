# Claude Custom Agents

A collection of custom agents for Claude Code.

## Agents

### Best Practices Researcher
**File:** `best-practices-researcher.md`

Expert technology researcher specializing in discovering, analyzing, and synthesizing best practices from authoritative sources. Uses Context7, Firecrawl, and web search to provide comprehensive, actionable guidance based on current industry standards.

**Use cases:**
- Researching best practices for any technology or framework
- Finding official documentation and community standards
- Discovering well-regarded examples from open source projects
- Getting up-to-date implementation guidance

**Model:** Haiku (fast and cost-effective)

## Usage

1. Copy the agent file to your `~/.claude/agents/` directory
2. The agent will be automatically available in Claude Code
3. Claude will invoke it when research tasks match the agent's description
